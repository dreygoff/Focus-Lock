# Focus Lock 🔒

**Focus Lock** — это Android-приложение для повышения продуктивности, которое помогает пользователям бороться с цифровой зависимостью. Оно позволяет блокировать выбранные приложения (например, социальные сети), перекрывая их кастомным экраном фокусировки при попытке запуска.

Проект разработан с использованием современного стека технологий Android, демонстрируя применение **Jetpack Compose**, **Clean Architecture** и реактивного программирования.

## 🛠 Технологический стек & Архитектура

Приложение построено на принципах **Чистой Архитектуры (Clean Architecture)** с разделением на независимые слои (`Data`, `Domain`, `Presentation`) и паттерне **MVVM**.

* **UI-фреймворк:** Jetpack Compose (полностью декларативный интерфейс без XML)
* **Асинхронность & Потоки данных:** Kotlin Coroutines & Реактивные потоки (Flow / StateFlow)
* **Внедрение зависимостей:** Dagger Hilt (`@HiltAndroidApp`, `@HiltViewModel`, модули с автоматическим управлением областями видимости)
* **Локальное хранилище:** Room Database (для сохранения списка заблокированных пакетов в реальном времени)
* **Интеграция с ОС:** `AccessibilityService` (Служба доступности) для системного трекинга окон + `WindowManager` для оверлеев.
